# 实验四：基于 LSTM 实现春联上联对下联

## 一、实验内容

本实验基于 LSTM 模型实现春联上联对下联任务，完成了数据读取、词表构建、模型训练、模型测试和实验报告整理。

## 二、目录说明

- `train.py`：训练脚本，支持设置训练轮数和设备
- `test.py`：测试脚本，输出测试损失和样例生成结果
- `dataprocessor.py`：数据读取、数据切分和词表构建
- `model.py`：LSTM 网络结构定义
- `demo.py`：基于 Flask 的简易演示程序
- `data/`：训练集和测试集数据
- `img/`：项目结构图和演示截图素材
- `save/`：训练后生成的模型、词表和损失曲线
- `artifacts/`：训练与测试日志
- `reports/build_experiment4_report.py`：实验报告生成脚本
- `reports/实验04_基于LSTM实现春联上联对下联_实验报告.docx`：最终实验报告

## 三、实验环境

- 操作系统：macOS
- Python：3.12.13
- PyTorch：2.11.0
- 训练设备：Apple Silicon MPS
- 训练轮数：10

## 四、运行命令

训练：

```bash
python train.py --epochs 10 --device mps
```

测试：

```bash
python test.py --device mps
```

## 五、实验结果

- 第 10 轮训练损失：`6.446218`
- 第 10 轮验证损失：`6.419461`
- 第 10 轮测试损失：`6.563898`
- 单独测试脚本输出损失：`6.515534`

模型生成样例：

- 上联：春回大地
  下联：春风风一
- 上联：福满人间
  下联：一万年风
- 上联：岁岁平安
  下联：一千里风

## 六、说明

1. 本实验按课程要求完成了 10 轮训练，并保存了训练日志、测试日志和损失曲线。
2. 仓库中保留了训练后的 `best_model.pt`，可以直接用于测试和演示。
3. 如果继续增加训练轮数，或改用带注意力机制的 Seq2Seq 模型，通常可以得到更自然的下联结果。
