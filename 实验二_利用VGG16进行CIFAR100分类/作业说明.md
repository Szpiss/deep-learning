# 实验二：利用 VGG-16 进行 CIFAR-100 分类

## 一、实验内容

本实验使用 `VGG-16` 网络对 `CIFAR-100` 数据集进行图像分类，完成了模型训练、模型测试、结果分析和实验报告整理。

## 二、目录说明

- `train.py`：训练脚本
- `test.py`：测试脚本
- `utils.py`：数据加载、网络选择和辅助函数
- `conf/`：实验配置文件
- `models/vgg.py`：VGG-16 网络结构定义
- `artifacts/vgg16_train_10epochs.log`：10 轮训练日志
- `artifacts/vgg16_test_best.log`：最佳权重测试日志
- `figures/`：训练损失曲线和测试准确率曲线
- `screenshots/`：训练终端截图和测试终端截图
- `reports/build_experiment2_report.py`：实验报告生成脚本
- `实验02_利用VGG16进行CIFAR100分类_实验报告.docx`：最终实验报告

## 三、实验环境

- 操作系统：macOS
- Python：3.12.13
- PyTorch：2.11.0
- Torchvision：0.26.0
- 训练设备：Apple Silicon MPS

## 四、实际运行命令

训练命令：

```bash
python train.py -net vgg16 -device mps -epochs 10 -b 128 -num-workers 0
```

测试命令：

```bash
python test.py -net vgg16 -device mps -weights checkpoint/vgg16/Tuesday_07_April_2026_14h_25m_58s/vgg16-10-best.pth -b 128 -num-workers 0
```

## 五、实验结果

- 第 10 轮测试准确率：`0.2678`
- Top-1 error：`0.7322`
- Top-5 error：`0.3840`
- 参数量：`34015396`
