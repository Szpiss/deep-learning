# dl_xor_lab

这个仓库用于存放课程实验与作业。

## 仓库结构

```text
dl_xor_lab/
├── 实验一_神经网络基础/
│   ├── xor_pytorch.py
│   └── 深度学习实验一_神经网络基础_实验报告.docx
├── 实验二_推荐系统_用户邻域TopN/
│   ├── 实验代码.py
│   ├── 实验报告.md
│   ├── 作业说明.md
│   └── 数据集/
└── README.md
```

## 各实验说明

- `实验一_神经网络基础/`
  保存异或问题实验代码和实验一报告。
- `实验二_推荐系统_用户邻域TopN/`
  保存基于用户邻域的 Top-N 推荐系统实验代码、实验说明、实验报告和数据集。 

## 说明

`实验二_推荐系统_用户邻域TopN/实验代码.py` 首次运行时会自动下载 `MovieLens 100K` 数据集到对应实验目录下的 `数据集/` 文件夹，因此仓库中默认不提交这批数据文件。
