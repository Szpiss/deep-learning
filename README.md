# dl_xor_lab

这个仓库用于存放课程实验与作业。

## 仓库结构

```text
dl_xor_lab/
├── 实验一_神经网络基础/
│   ├── xor_pytorch.py
│   └── 深度学习实验一_神经网络基础_实验报告.docx
├── 实验二_利用VGG16进行CIFAR100分类/
│   ├── train.py
│   ├── test.py
│   ├── utils.py
│   ├── conf/
│   ├── models/
│   ├── artifacts/
│   ├── figures/
│   ├── screenshots/
│   ├── reports/
│   ├── 作业说明.md
│   └── 实验02_利用VGG16进行CIFAR100分类_实验报告.docx
└── README.md
```

## 各实验说明

- `实验一_神经网络基础/`
  保存异或问题实验代码和实验一报告。
- `实验二_利用VGG16进行CIFAR100分类/`
  保存基于 VGG-16 的 CIFAR-100 图像分类实验代码、实验报告、训练测试日志、曲线图和终端截图。

## 说明

本仓库当前专门用于保存深度学习课程实验。原先的推荐系统实验已迁移到独立仓库 `recommend-system`：
<https://github.com/Szpiss/recommend-system>
