## 各实验说明

- `实验一_神经网络基础/`
  保存异或问题实验代码和实验一报告。
- `实验二_利用VGG16进行CIFAR100分类/`
  保存基于 VGG-16 的 CIFAR-100 图像分类实验代码、实验报告、训练测试日志、曲线图和终端截图。
- `实验四_基于LSTM实现春联上联对下联/`
  保存基于 LSTM 的春联自动对下联实验代码、数据、训练结果、测试日志和实验报告。
